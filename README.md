# 🌱 Crop Calendar and Yield Optimizer

A smart web application that helps farmers and agriculture enthusiasts plan crop cultivation efficiently. The application provides crop recommendations, harvesting schedules, soil recommendations, fertilizer suggestions, pesticide recommendations, and crop yield optimization based on the selected crop and sowing date.

## 🚀 Live Demo

🔗 https://resilient-dusk-83d89f.netlify.app/

## 📌 Features

- 🌾 Select crop type
- 📅 Enter sowing date
- ⏳ Calculate expected harvest date
- 🌱 Soil type recommendations
- 🧪 Fertilizer recommendations
- 🐛 Pesticide recommendations
- 💧 Irrigation guidance
- 📈 Crop optimization plan
- 📱 Responsive user interface

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/niharikapanuganti/Crop-Calendar-Yield-Optimizer.git

Go to project folder

```bash
cd Crop-Calendar-Yield-Optimizer
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
